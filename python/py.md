### 2.4. 코드블록

들여쓰기가 동일한 코드들의 집합이다.

~~~python
if 조건:
    명령 # 코드블록
    명령
    명령
명령     # 블록 외부의 명령
명령
~~~

~~~python
if a == 3:
    print('3') # 코드블록
    print('three')
    print('3과 같습니다')
~~~
